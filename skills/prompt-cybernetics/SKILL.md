# Prompt Engineering × Cybernetics Skill

> **Meta-skill for evolving prompt infrastructure and daily optimization protocols**

## Overview

This skill creates a 10-agent research swarm that maps the full landscape of prompt engineering, blends it with cybernetic principles (feedback loops, self-regulation, emergence), and produces a **Daily Prompt Optimization System** for maximum evolution velocity.

## The 10 Agents

| # | Agent | Domain | Output |
|---|-------|--------|--------|
| 1 | **Prompt Archaeologist** | Historical patterns | Evolution of prompting (GPT-2 → GPT-5, Claude 1 → 4.5) |
| 2 | **Structural Engineer** | Token-level mechanics | Optimal token placement, attention hacking, position bias |
| 3 | **Context Window Optimizer** | Long-context strategies | Compression, summarization, hierarchy, retrieval patterns |
| 4 | **Cybernetic Feedback Specialist** | Self-regulation loops | Prompts that monitor and adjust their own output |
| 5 | **Emergence Hunter** | Non-linear effects | Prompts that trigger phase transitions in model behavior |
| 6 | **Multi-Modal Architect** | Cross-modal prompting | Text ↔ Image ↔ Code ↔ Structured data flows |
| 7 | **Constraint Designer** | Bounded creativity | Using limitations to increase output quality |
| 8 | **Persona Systems Engineer** | Identity frameworks | When/how to use system prompts vs inline personas |
| 9 | **Metacognitive Layer Designer** | Self-reflection prompts | Getting models to reason about their own reasoning |
| 10 | **Evolutionary Optimizer** | Continuous improvement | Prompts that improve themselves through feedback |

## Daily Output Protocol

### The 1+4 System

```
DAILY_MASTER_PROMPT (1x/day, morning)
├── High-leverage strategic prompt
├── Targets: Evolution, breakthrough, paradigm shift
└── Fed to OpenClaw main session

DAILY_SUB_PROMPTS (4x/day, every 6 hours)
├── Tactical optimization prompts
├── Targets: Specific projects, incremental gains, maintenance
└── Fed to relevant subagents/work sessions
```

## Usage

```bash
# Initialize the 10-agent swarm
clawd prompt-cybernetics init

# Run deep dive research (generates knowledge base)
clawd prompt-cybernetics research --depth comprehensive

# Generate today's prompts
clawd prompt-cybernetics daily

# List all prompt patterns discovered
clawd prompt-cybernetics patterns

# Test a prompt pattern
clawd prompt-cybernetics test --pattern emergence_hunting --input "..."
```

## Key Principles (Cybernetic Lens)

1. **Feedback Loops**: Every prompt includes a mechanism for self-correction
2. **Variety Engineering**: Controlled randomness prevents mode collapse
3. **Ashby's Law**: Sufficient internal variety to match external complexity
4. **Autopoiesis**: Prompts that maintain and reproduce their own structure
5. **Second-Order Cybernetics**: The observer is part of the system
